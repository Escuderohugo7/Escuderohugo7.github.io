Aplicaciones Web

RESUMEN:

Las Aplicaciones Web es una asignatura fundamental en primer año de grado medio que introduce al estudiante en el desarrollo y uso de aplicaciones accesibles a través de navegadores web. En esta materia, se aprenden los conceptos básicos de la estructura cliente-servidor, los lenguajes de marcado como HTML, el diseño estilizado con CSS y la interacción básica mediante JavaScript. El alumno descubre cómo funcionan las páginas web dinámicas, la importancia de la usabilidad y el diseño responsive para adaptarse a distintos dispositivos. También se trabajan aspectos prácticos como la creación de formularios, la integración de contenido multimedia y el uso de herramientas de desarrollo como editores de código o inspectores de navegador. Además, se introducen nociones sobre seguridad web, protección de datos y publicación de sitios en servidores. Combinando teoría con ejercicios prácticos, esta asignatura proporciona las bases para entender el funcionamiento de la web moderna y prepara al alumno para profundizar en desarrollo front-end o trabajar con CMS en estudios posteriores.
